## Avatars

Simple intuitive avatar components of all primary sizes

Sizes - `sm` `md` `lg`

<section style="display: flex; margin: 2rem 0">
  <div class="cl-avatar avatar-sm" style="margin-right: 1.6rem"></div>
  <div class="cl-avatar avatar-md" style="margin-right: 1.6rem"></div>
  <div class="cl-avatar avatar-lg" style="margin-right: 1.6rem"></div>
  <div class="avatar-active">
    <div class="cl-avatar avatar-lg"></div>
  </div>
</section>

### HTML version

```html title=src/post-card.html
<div class="cl-avatar avatar-sm"></div>
<div class="cl-avatar avatar-md"></div>
<div class="cl-avatar avatar-lg"></div>
<div class="avatar-active">
  <div class="cl-avatar avatar-lg"></div>
</div>
```

### React version

| Prop     | Value       | Default |
| -------- | ----------- | ------- |
| size     | sm, md, lg  | md      |
| isActive | true, false | false   |

```jsx
<Avatar size="sm"/>
<Avatar size="md"/>
<Avatar size="lg"/>
<Avatar size="lg" isActive={true}/>
```
